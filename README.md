# 🤖 AI Mock Interview Bot

An intelligent mock interview platform powered by **Llama3 AI** that generates 
role-specific interview questions, evaluates answers in real-time, and provides 
detailed feedback with scores.

## ✨ Features

- 🎯 **Role-Based Questions** — AI generates unique questions for each job role
- 🧠 **Real-Time AI Evaluation** — Every answer scored and analyzed instantly
- 📊 **Detailed Feedback** — Score, strengths, weaknesses, and ideal answers
- 💾 **Interview History** — All sessions saved to database permanently
- 🆓 **100% Free** — Runs on local Ollama AI, zero API costs
- 📱 **Responsive UI** — Works on desktop and mobile

---

## 🛠️ Tech Stack

| Layer | Technology | Purpose |
|---|---|---|
| Frontend | React + Vite + Tailwind | Beautiful responsive UI |
| Backend | Python + FastAPI | REST API server |
| AI Engine | Ollama + Llama3 | Free local AI model |
| Database | SQLite + SQLAlchemy | Interview history storage |
| Deployment | Vercel + Render | Free cloud hosting |
